golang Cookbook
====================

Requirements
============

Platform
--------

Tested on:

* Debian GNU/Linux 6.0

**Notes**: This cookbook has been tested on the listed platforms. It
  may work on other platforms with or without modification.

Usage
-----
#### golang::default

Just include `golang` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[golang]"
  ]
}
```
